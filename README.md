AsteroidsJS
==========

Old school asteroids with HTML5 Canvas

[Try it out!](http://pclements12.github.io/AsteroidsJS/asteroids.html)

# Build

`npm run build`

# Watch

`npm run watch`

# Local Dev Server

`npm run serve`

# Local Dev Server with Watch

`npm start`
(still requires reload in browser for changes)
